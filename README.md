# Rethinking Front End Developmet With Elm

Code, slides, and materials for my Elm talk.

Slides are built for Deckset.

To run code

```
$ npm install -g elm,
$ npm install -g elm-live
```

Then, in `code/` folder

```
$ elm package install
$ elm live calculator.elm --open
```
